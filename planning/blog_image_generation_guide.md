## Generation Tips

### For AI Image Generators (DALL-E, Midjourney, Stable Diffusion)

1. **Start with the base prompt** provided above
2. **Add style modifiers** if needed:
   - "4k resolution, high detail"
   - "professional marketing visual"
   - "dark mode UI aesthetic"
   - "no text overlays"

3. **Iterate as needed:**
   - If too busy, add: "minimalist, clean composition"
   - If too generic, add: "unique, creative, editorial quality"
   - If wrong colors, emphasize: "black background #0a0a0a, crimson red #DC143C accents only"

4. **Common negative prompts:**
   - "no text, no words, no letters, no watermarks"
   - "no people, no faces, no hands"
   - "no stock photo aesthetic"
   - "no brand logos"

### File Naming Convention
Save all files as lowercase with hyphens:
- `linkedin-algorithm.png`
- `email-marketing.png`
- etc.

### After Generation
1. Verify dimensions are 1600 x 900 px (resize if needed)
2. Optimize file size (target under 500KB)
3. Place in `/public/images/blog/`
4. Test display on blog post pages
